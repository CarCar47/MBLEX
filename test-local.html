Development Console Commands

Once the app is running, open browser DevTools and try these commands:

  • MBLEXApp.getDebugInfo() - Get app status
  • MBLEXApp.navigateToStudy() - Go to study area
  • MBLEXApp.navigateToTest() - Go to test generator
  • MBLEXApp.navigateToSimulation() - Go to simulations
  • MBLEXApp.navigateToChatbot() - Go to AI tutor

Keyboard Shortcuts

  • Ctrl+M - Return to main menu
  • Ctrl+1 - Study area
  • Ctrl+2 - Test generator
  • Ctrl+3 - Simulations
  • Ctrl+4 - AI tutor
  • Escape - Go back
  • 1-4 - Quick navigation from main menu
